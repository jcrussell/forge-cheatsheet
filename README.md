# Forge Window Manager - Keyboard Cheatsheet

A concise, printable keyboard reference for the [Forge](https://github.com/forge-ext/forge) tiling window manager extension for GNOME Shell.

## About

This cheatsheet provides a quick reference for all Forge keyboard shortcuts, organized by category including focus & navigation, window lifecycle, tiling layouts, precision resizing, workspaces, and troubleshooting.

## Getting the PDF

Download the latest version from the [Releases page](../../releases).

Alternatively, build it yourself with XeLaTeX:

```bash
make
```

## About Forge

Forge is a tiling window manager extension for GNOME Shell that brings i3/sway-like window management to GNOME. For installation, documentation, and source code, visit the [official Forge repository](https://github.com/forge-ext/forge).
